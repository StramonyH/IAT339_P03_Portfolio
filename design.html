Design Process Analysis

Introduce

This is the website design is about the cake shop. Use the website to display the products and services of the cake shop, so that more people can freely buy products on the web page.

Design Style

In this website, the main colors are pink and coffee. Pink is used as the background color and coffee color is used as the text color to better highlight the content. A relatively wide margin is used between each section to make the website look less crowded. The overall design style is biased towards cuteness, and the font also uses a rounder font. Because most of the people buying cakes are mainly women, it can attract more customers to browse the website.

Analyze

The cake website needs to reflect what services and products can be provided,and we must design a service list with a clear hierarchical structure. For example, use a carousel in homepage can let the users able to see the new products once they enter the site. At the same time, several important sections have been designed on the homepage and can be linked to subpages for more detailed understanding. In the product list, On the product detail page, you can not only see the detailed introduction of this cake or drink, but also the recommendations of other products. We also need to consider that the website can be displayed perfectly on different screens,so we also need to have different responsive designs. For the screen that is too small, we need to change some pictures and menus.
