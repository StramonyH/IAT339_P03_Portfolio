Website design

Introduction

This is the website design about a simulated cake shop. This website is used to showcase the products and services provided by the cake shop, as well as allow people to freely learn about the brand and purcahse products on the web page.

Technology Used

HTML5, CSS, JavaScript: to create the whole website and interaction functions.

Adobe Illustrator: to create cake posters.

Design Style

In this website, the main colors are pink and coffee. Pink is used as the background color and coffee color is used as the text color to better highlight the content. A relatively wide margin is used between each section to make the website look less crowded. The overall design style is biased towards cuteness, and the font also uses a rounder font. Because most of the people buying cakes are mainly female (according to our previous customer research), so it can attract more customers to browse the website.

Analyze

This cake website needs to reflect what services and products are provided, and we must design a service list with a clear hierarchy. For example, using a carousel in homepage can let the users able to see the new products once they enter the site. At the same time, several important sections have been designed on the homepage and can be linked to subpages for more detailed understanding. On the product list page and the product detail pages, you can see not only the detailed introduction of this cake or drink, but also the recommendations of other products. We also need to consider that the website can be displayed perfectly on different screens,so we also need to have different responsive designs. For the screen that is too small, we need to change some pictures and menus.

Feedback

This project was completed by me and another partner. I am quite satisfied with this project, and I learned some interesting interactive designs, such as how the carousel can be used. At the same time, the company's service navigation and product list are given in detail which means the contents are also well.
