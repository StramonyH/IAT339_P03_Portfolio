Game design

Introduction

This game project is called Magic Kitchen. It is to simulate the process of cake making. The whole cake is made in the kitchen. The goal is people need to according to instruction to finish a delicious cake.

Technology Used

Eclipse IDE: to create the process of interaction.

Adobe Illustrator: to create the scenes and objects.

Design Style

In this interactive scene, the overall style is dominated by blue, giving the user a visual softness. The start interface and the end interface are mainly pink. In the entire interaction, sound elements are also added to make the scene more realistic.

Analyze

In order to reflect the authenticity, when entering the interface at the beginning, the user needs to turn on the lights in the scene to start the cake making process, the entire interaction process is carried out in accordance with the actual cake making steps and the user will be prompted to do the next step at the top of the interface what.The main interaction method is user can use mouse drag and mouse click to finish interaction. When user click or drag the image, image will change the state(change image) and into next step. Like if user put the cake into oven, the next image is cake is baked.

Feedback

In this project, I think the better point is that it fully embodies the entire interactive process of making cakes, which is more realistic. After completing this project, I learned how to combine code and pictures to reflect an interactive scene. I hope to add more animations to this project.
